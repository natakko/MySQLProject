Отчет по продажам (T-SQL)
Хранимая процедура для расчета ключевых бизнес-показателей и анализа продаж.

Что реализовано в коде:
Гибкий фильтр по группам: Процедура принимает список категорий товаров через запятую. Внутри реализован парсинг строки через цикл WHILE, что позволяет фильтровать отчет сразу по нескольким группам.
Работа с форматами данных: Входные параметры типа DATE автоматически конвертируются в INT (ггггммдд) для корректной связи с таблицей фактов.
Расчет аналитики:
Вычисление средней цены закупки, маржи и наценки (с очисткой от НДС 20%).
Определение доли продаж каждого товара в общем обороте через оконную функцию OVER().
Безопасность расчетов: Использование NULLIF при делении, чтобы исключить ошибки при нулевых значениях в данных.

Стек технологий:
Среда разработки: SQL Server Management Studio (SSMS) 22.
Язык: T-SQL (Microsoft SQL Server).
Использованные инструменты: Временные таблицы (#temp), INNER JOIN, оконные функции, CASE и обработка строк.

Как запустить:
Выполните скрипт создания процедуры в базе данных (например, TestTaskDB).
Для запуска отчета используйте команду:
EXEC [dbo].[sp_report_1] 
    @date_from = '2017-12-28', 
    @date_to = '2017-12-28', 
    @good_group_name = N'Биологически активные добавки'
